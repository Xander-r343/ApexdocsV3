# About Apex
---
## What is Apex?
Apex Pathing is a passion project developed by multiple individuals from FIRST tech Challenge teams around the world.
Our goal is to provide an advanced pathing experience for beginner programmers. We have support for many drivetrains sucb as mecanum,
x-drive, tank, and swerve. All you need is a drivetrain, a localization solution, and a knowledge of Java programming in FTC!

## Ready to get started?
Check out the [installation guide](/docs/installation-guide) to get Apex up and running.
